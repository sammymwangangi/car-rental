## Car Rental

Car rental management system.

### Installation
1. clone
2. cd car-rental
3. composer install
4. copy .env.example .env
5. php artisan key:generate
6. php artisan storage:link
7. php artisan migrate --seed
8. npm install && npm run dev
9. php artisan serve
